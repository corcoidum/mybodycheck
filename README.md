# 마이 바디 체크

폰에서 홈 화면 앱처럼 열어 매일 두통, 어지러움, 수면, 컨디션을 기록하는 단일 HTML 건강 체크 도구입니다.

## 휴대폰에서 쓰는 방법

1. 이 폴더의 파일들을 정적 웹 호스팅에 올립니다. GitHub Pages, Cloudflare Pages, Netlify 같은 HTTPS 주소면 됩니다.
2. 휴대폰 브라우저에서 해당 주소를 엽니다.
3. Android Chrome은 `홈 화면 설치` 버튼을 누릅니다. iPhone Safari는 공유 버튼에서 `홈 화면에 추가`를 선택합니다.
4. 이후 홈 화면의 `바디체크` 아이콘으로 열면 기록과 히스토리를 바로 확인할 수 있습니다.

기록은 사용하는 휴대폰 브라우저 안에 저장됩니다. 기기를 바꾸거나 브라우저 데이터를 지우기 전에는 앱 안의 `백업` 버튼으로 JSON 파일을 보관하고, 새 기기에서 `복원`으로 가져오세요.
